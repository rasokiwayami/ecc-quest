# Quest 02 — 実装計画を立てる

**章**: 第1章「小さなスクリプト」  
**スキル**: `/plan` を 1回  
**報酬**: 80 XP

---

## ストーリー

PRDで「何を作るか」が決まった。次は「どう作るか」を決める番だ。

計画なしにコードを書き始めると、途中で迷子になる。  
`/plan` は実装の順序・依存関係・リスクを整理し、進むべき道を示してくれる。

---

## 指示

1. Claude Code のチャットで以下を実行する:

```
/plan
```

2. Quest 01 で作ったPRDの内容をもとに、実装計画を立てる  
3. 計画を確認し、必要なら修正する  
4. 完了したら以下を実行してXPを受け取る:

```bash
node scripts/check.js record plan
```

---

## ヒント

- 計画は完璧でなくて良い。まずざっくりとした方向性があれば十分
- フェーズ分けを意識する（Phase 1: 動くもの、Phase 2: 整える）
- リスクを1つ以上挙げることで計画の質が上がる

---

## クリア条件

`/plan` を 1回実行 → `node scripts/check.js record plan` でXP付与
