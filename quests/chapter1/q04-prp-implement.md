# Quest 04 — コードを実装する（2回）

**章**: 第1章「小さなスクリプト」  
**スキル**: `/prp-implement` を 2回  
**報酬**: 最大200 XP（初回100 + 2回目×0.7）

---

## ストーリー

テストが揃った。残りのコードを仕上げる時だ。

`/prp-implement` はPRDと計画を読み込み、残タスクを整理しながら実装を進める。  
ただ「コードを書いて」と頼むのと違い、文脈を保ちながら動く。

2回使うことで「最初の実装」と「改善サイクル」を体験する。

---

## 指示

1. Claude Code のチャットで以下を実行する:

```
/prp-implement
```

2. 指示に従って実装を進める  
3. 完了したら以下でXPを記録する:

```bash
node scripts/check.js record prp-implement
```

4. 機能を追加・改善してもう1回繰り返す（計2回）

---

## ヒント

- 1回目: 主要機能の実装
- 2回目: エラー処理・エッジケースの対応、または次の機能
- 実装後はすぐに `record` を実行する習慣をつける
